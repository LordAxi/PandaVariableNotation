# PandaVariableNotation

## Syntax

Declerations: {type}.new
Redeclarations: {type}.edit

### Types

s = string
i = integer
f = float
b = boolean

### Whitespace

This Notation is heavily based on Whitespace to make reading easier.

### Examples

s.new name = pnv
i.new year_created = 2025
f.new age_in_days = 1.5
